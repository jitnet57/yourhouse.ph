# GitHub 사용법 및 TODO 리스트

이 문서는 GitHub를 사용하는 방법과 주요 할 일(TODO)을 정리한 가이드입니다.

## 1. GitHub란?
- GitHub는 소스 코드 버전 관리를 위한 플랫폼입니다.
- 협업, 코드 리뷰, 이슈 관리, 배포 자동화 등 다양한 기능을 제공합니다.

## 2. 기본 사용법
1. 저장소(Repository) 생성
2. 로컬에서 git clone으로 복제
3. 파일 추가/수정 후 커밋
4. 원격 저장소로 push
5. Pull Request(PR)로 코드 리뷰 요청
6. Merge로 코드 반영

## 3. 주요 명령어
- 저장소 복제: `git clone <URL>`
- 변경 사항 확인: `git status`
- 파일 추가: `git add .`
- 커밋: `git commit -m "메시지"`
- 원격 저장소로 푸시: `git push origin main`
- 브랜치 생성: `git branch <브랜치명>`
- 브랜치 전환: `git checkout <브랜치명>`
