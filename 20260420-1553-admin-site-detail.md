# 🏢 K-IREA 어드민 대시보드 상세 안내 (Admin Site Detail)

**K-IREA(Korea Integrated Real Estate AI)** 플랫폼의 핵심 관리 도구인 어드민 대시보드에 대한 상세 가이드입니다. 이 대시보드는 부동산 에이전트와 관리자가 플랫폼의 모든 활동을 제어하고 분석할 수 있도록 설계되었습니다.

---

## 1. 📋 개요 (Executive Summary)

### **비전 (Vision)**
부동산 에이전트, 매니저, 관리자가 다음 작업을 효율적으로 수행할 수 있는 **통합 관리 환경**을 제공합니다:
- **매물 관리:** AI 기반 데이터 검증 및 등록
- **리드 추적:** 잠재 고객 파이프라인 및 전환 관리
- **AI 에이전트 모니터링:** 24/7 작동하는 AI 챗봇 및 보이스봇 상태 감시
- **커뮤니케이션 제어:** SMS, WhatsApp, 음성 상담 내역 통합 관리
- **성과 분석:** ROI(투자 대비 수익) 및 시장 트렌드 데이터 리포팅

### **주요 목표 (Success Metrics)**
- **정확성:** AI 검증을 통한 오허위 매물 0% 지향
- **응답성:** 모든 대시보드 반응 속도 300ms 이내
- **보안:** 역할 기반 접근 제어(RBAC) 및 보안 인증 강화

---

## 🏗️ 2. 기술 아키텍처 (Architecture)

대시보드 서비스는 현대적인 웹 기술 스택을 사용하여 구축되었습니다.

- **Frontend:** Next.js 15+, React 18, Tailwind CSS (반응형 다크 모드 지원)
- **Backend:** FastAPI (Python), RESTful API 아키텍처
- **Database:** PostgreSQL (Supabase를 통한 클라우드 관리)
- **Authentication:** NextAuth.js (JWT 기반 보안 인증)
- **Real-time:** Webhook 및 실시간 데이터 스트리밍

---

## 📊 3. 핵심 기능 (Core Features)

### **1. 대시보드 및 분석 (Dashboard & Analytics)**
- **주요 지표:** 활성 매물 수, 신규 리드, 전환율, 이번 달 매출, AI 응답 시간
- **시각화 차트:** 리드 파이프라인(퍼널), 전환 트렌드, 지역별 인기 매물 등
- **최근 활동 피드:** 실시간으로 발생하는 신규 문의 및 계약 체결 알림

### **2. 매물 관리 (Property Management)**
- **테이블 뷰:** 주소, 가격, 상태(판매 중/완성/초안), AI 검증 점수 표시
- **상세 편집:** 사진 갤러리 관리, AI 매물 분석 리포트 확인, 방문 로그 추적
- **일괄 작업:** 대량 데이터 임포트/익스포트, 매물 상태 일괄 변경

### **3. 리드 및 파이프라인 (Lead Management)**
- **칸반(Kanban) 보드:** 신규 -> 연락 중 -> 관심 있음 -> 협상 중 -> 계약 완료 단계로 관리
- **AI 채팅 기록:** AI 챗봇과 고객 간의 전체 대화 스크립트 확인 및 필요 시 상담원 개입
- **자동 알림:** 팔로우업이 필요한 리드에 대한 리마인더 알림

### **4. AI 에이전트 제어 센터 (AI Agent Controls)**
- **에이전트 설정:** 인사말, 말투(공식적/친절함), 활동 시간, 답변 불가 시 대응 로직 설정
- **통합 채널:** 음성(Voice), SMS, WhatsApp 채널별 활동 모니터링
- **성능 측정:** 고객 만족도, 평균 해결 시간, 지식 베이스 정확도 측정

---

## 🔐 4. 보안 및 권한 (Security)

- **Admin (관리자):** 모든 대시보드 접근, 사용자 관리, 시스템 설정 변경 권한
- **Manager (매니저):** 팀 성과 관리, 에이전트 활동 모니터링 권한
- **Agent (에이전트):** 본인이 등록한 매물 및 리드 관리, 성과 확인 권한
- **Viewer (뷰어):** 리포트 조회 전용 권한 (수정 불가)
